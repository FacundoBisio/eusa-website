# EUSA Venture Studio – Landing Website

The website is active in: https://eusa-partners.com/

This repository contains the source code for the official landing page of **EUSA Venture Studio**, built in HTML, CSS (Bootstrap 5) and vanilla JavaScript, deployed on a traditional shared hosting environment. Developed by Facundo Bisio

The website includes:
- Fully responsive layout (mobile-first)
- Hero section with value proposition
- How We Help / Sprint details
- Why EUSA section with feature cards
- Leadership team profiles
- Contact form with PHP backend (AJAX)
- Smooth animations and subtle transitions
- Clean componentized structure (`assets/css`, `assets/img`)

---

## 🚀 Tech Stack

- **HTML5**  
- **CSS3** + Bootstrap 5  
- **JavaScript (ES6)**  
- **PHP** (used only for email sending)  
- **Shared Hosting environment** (DirectAdmin / FTP upload)
